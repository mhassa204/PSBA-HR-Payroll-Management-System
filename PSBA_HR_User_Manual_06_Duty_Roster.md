# PSBA HR Management System - User Manual

## Part 6: Duty Roster Management

---

## Overview

The Duty Roster module allows location managers to create and manage work schedules for employees at their assigned locations. Rosters define working days, shifts, and time slots for each employee.

**Key Features:**

- Create duty rosters for locations
- Assign employees to specific time slots
- Set weekly schedules
- Define shift timings
- Approve roster changes
- View roster assignments

**Who Can Use This Module:**

- Location Managers (with location permissions)
- HR Staff (Establishment role)
- Users with roster management permissions

---

## Understanding Duty Rosters

### What is a Roster?

A duty roster is a work schedule that specifies:

- **Which employees** work on which days
- **What times** they work each day
- **Where** they are assigned (if multiple locations)
- **Special assignments** or notes

### Roster Terms

- **Schedule Period:** The time range the roster covers (e.g., "December 2024")
- **Employee Entry:** One employee's schedule for the roster period
- **Day Schedule:** The schedule for one day of the week
- **Time Slot:** Start and end times for a shift
- **Weekly Off:** Days when an employee does not work

### Approval Status

Rosters have approval statuses:

- **Pending:** Awaiting approval
- **Approved:** Approved and active
- **Rejected:** Not approved

## Creating a New Duty Roster

### Accessing Roster Creation

1. From the main menu, click **"Duty Roster"**
2. Click **"Create Roster"** or the "+" button
3. The system opens the roster creation form

### Filling Out Roster Information

#### Step 1: Basic Information

1. **Title** (Optional)

   - Enter descriptive title
   - Example: "Security Roster - December 2024"
   - This helps identify rosters later

2. **Valid From**

   - Select start date from calendar
   - First day the roster becomes effective

3. **Valid To**
   - Select end date from calendar
   - Last day the roster is effective
   - Ensure it's after the start date

#### Step 2: Understanding the Roster Interface

The roster shows all employees assigned to your location, organized by:

- **Role Tags:** Grouped by job category
- **Employee Name:** Individual employees

For each employee, you'll see a grid with:

- **Day Columns:** Monday through Sunday
- **Time Fields:** For entering work times
- **Action Buttons:** To set schedule types

#### Step 3: Setting Employee Schedules

For each employee and each day of the week:

##### Regular Work Day

1. Click on the day cell for the employee
2. Set schedule type to "Time"
3. Enter:
   - **Time From:** Start time (e.g., 09:00)
   - **Time To:** End time (e.g., 17:00)
   - **Location:** Optional location identifier

##### Weekly Off

1. Click on the day cell
2. Select schedule type as "Weekly Off"
3. That day is marked as off for the employee

##### Half Day

1. Click on the day cell
2. Set schedule type to "Half Day"
3. Enter partial time:
   - **Time From:** Start time
   - **Time To:** End time (shorter duration)

##### Leave

1. Click on the day cell
2. Select schedule type as "Leave"
3. Day is marked as leave

#### Step 4: Setting Collective Weekly Off

For employees who have the same off day(s) every week:

1. Find the employee row
2. Click on weekly off settings
3. Enable collective weekly off
4. Set:
   - **From Day:** First day of off (e.g., Saturday)
   - **To Day:** Last day of off (e.g., Sunday)
5. This applies to all weeks in the roster

#### Step 5: Adding Remarks

For any employee, you can add remarks:

1. Click "Remarks" for the employee
2. Enter notes
3. Notes apply to that employee's entire schedule

#### Step 6: Saving the Roster

1. Review all schedules
2. Verify dates and times
3. Click **"Save"** or **"Create Roster"**
4. Roster is created with status "Pending"
5. You'll be redirected to roster list

## Viewing Rosters

### Accessing Roster List

1. Go to Duty Roster from main menu
2. View list of all rosters

### Understanding Roster List

The list displays:

- **Title:** Roster name
- **Location:** Where roster applies
- **Valid From:** Start date
- **Valid To:** End date
- **Created By:** Who created it
- **Status:** Pending, Approved, or Rejected
- **Actions:** View, Edit, Delete options

### Filtering Rosters

You can filter rosters by:

- **Location:** View rosters for specific location
- **Date Range:** Rosters within dates
- **Status:** Approved, Pending, Rejected
- **Created By:** Your rosters vs. all rosters

## Viewing Roster Details

### Opening a Roster

1. Click on roster title or "View" button
2. Roster details page opens

### Roster Details Page

Shows complete roster information:

- **Header Information**

  - Title, dates, location
  - Creator, creation date
  - Current status

- **Employee Schedule Table**

  - All employees
  - Day-by-day breakdown
  - Time slots for each day
  - Visual grid layout

- **Summary**
  - Total employees
  - Schedule statistics
  - Coverage information

## Editing a Roster

### Accessing Edit Mode

1. Open roster
2. Click **"Edit"** button
3. Edit mode opens

### Frame-by-Frame Modifications

Make changes the same way as creating:

1. Update any employee's schedule
2. Modify time slots
3. Change weekly off days
4. Update remarks
5. Adjust dates if needed

### Saving Changes

1. Review modifications
2. Click **"Save Changes"**
3. Updates are saved
4. Roster may need re-approval if approved

---

## Approving Rosters

### Accessing Approval Queue

Rosters pending approval are shown to authorized approvers.

### Reviewing for Approval

1. Click on roster
2. Review complete schedule
3. Check:
   - All days covered
   - Adequate staffing
   - Reasonable hours
   - No conflicts

### Approving

1. Click **"Approve"** button
2. Add comments if needed
3. Submit approval
4. Roster status changes to "Approved"

### Rejecting

1. Click **"Reject"** button
2. Enter rejection reason (required)
3. Submit rejection
4. Creator is notified

## Understanding Roster Types

### Fixed Schedule

- Same schedule every week
- Consistent working hours
- Predictable assignments

### Rotating Schedule

- Schedules change between weeks
- Employees rotate shifts
- More complex to manage

### Flexible Schedule

- Varies by day
- Different patterns
- Location-specific needs

## Common Tasks

### Task: Create Monthly Roster

1. Go to Create Roster
2. Set title as "Month Year Roster"
3. Set valid from/to as start and end of month
4. Assign all employees their schedules
5. Save roster

### Task: Assign Same Schedule to Multiple Employees

1. While creating roster
2. Set one employee's schedule
3. Copy the pattern
4. Apply to similar employees
5. Fine-tune as needed

### Task: Change Employee's Day Off

1. Open roster
2. Click Edit
3. Find employee
4. Change the off day
5. Save changes

### Task: Add Employee to Roster

1. Open roster
2. Click Edit
3. Scroll to bottom
4. Click "Add Employee" (if available)
5. Set schedule
6. Save

### Task: View Employee's Schedule

1. Open roster
2. Find employee in list
3. View their row
4. See all weekdays
5. Check time slots

## Location-Based Rosters

### Understanding Location Context

- Rosters are tied to your location
- You can only create rosters for your assigned location
- Location managers see only their location's rosters

### Multi-Location Organizations

If managing multiple locations:

- Each location has separate rosters
- Create roster per location
- Switch between locations as needed

## Best Practices

1. **Plan in Advance:** Create rosters before period starts
2. **Balance Workload:** Distribute shifts fairly
3. **Cover All Days:** Ensure adequate staffing
4. **Document Changes:** Use remarks for modifications
5. **Regular Updates:** Keep rosters current
6. **Communication:** Share schedules with employees
7. **Compliance:** Follow labor regulations
8. **Review Regularly:** Check roster effectiveness

## Troubleshooting

### Cannot Create Roster

- Verify you're location manager
- Check location assignment
- Ensure proper permissions
- Contact administrator

### Can't See Employees in Roster

- Check employee is assigned to location
- Verify employment status
- Ensure location matches
- Refresh page

### Changes Not Saving

- Check all required fields
- Verify valid dates
- Ensure no conflicts
- Try saving again

### Roster Not Showing

- Check filters applied
- Verify date range
- Check location context
- Refresh list

## Important Notes

### Roster Validity

- Rosters are valid for specified date range
- Outside range, roster is inactive
- Create new rosters for new periods

### Approval Requirement

- Rosters typically need approval
- Work with approvers for quick approval
- Communicate changes in advance

### Schedule Flexibility

- Rosters can be modified
- Changes may require re-approval
- Keep employees informed

### Integration with Attendance

- Rosters link to attendance tracking
- Attendance is checked against rosters
- Discrepancies may be flagged
