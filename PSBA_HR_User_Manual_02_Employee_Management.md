# PSBA HR Management System - User Manual

## Part 2: Employee Management

---

## Overview

The Employee Management module allows you to view, create, and manage employee records in the system. This module is essential for maintaining accurate information about all staff members.

**Who Can Use This Module:**

- HR Staff (Establishment role)
- Administrators
- Users with employee management permissions

---

## Viewing Employees

### Accessing the Employee List

1. From the main menu, click on **"Employees"**
2. The system will display the **"All Employees"** page
3. You'll see a table listing all employees with their basic information

### Understanding the Employee List

The employee list table shows:

- **Employee ID:** Unique identification number
- **Full Name:** Complete name of the employee
- **CNIC:** National ID number
- **Email:** Official email address
- **Status:** Current employment status (Active/Inactive)
- **Department:** Current department assignment
- **Designation:** Current job title
- **Location:** Assigned workplace location

### Searching and Filtering Employees

1. **Use the Search Box**

   - Type any part of an employee's name, ID, or email
   - Results will filter automatically as you type

2. **Sort Columns**

   - Click on column headers to sort ascending/descending
   - Useful for organizing large lists

3. **Navigate Pages**
   - Use the pagination controls at the bottom
   - Click page numbers or next/previous buttons

#### Current Employment

## Creating a New Employee

**Note:** Only users with "Create Employee" permission can add new employees.

### Accessing the Create Employee Form

1. Click on **"Employees"** in the menu
2. Click **"Add Employee"** or the "+" button
3. The system will open the employee creation form

### Filling Out the Form

The form is divided into multiple sections. Complete each section:

#### Step 1: Personal Information

**Required Fields:**

- **Employee ID:** Unique identifier (e.g., EMP001)
- **Full Name:** Complete legal name
- **Email:** Official email address (must be unique)

**Optional Fields:**

- Father's/Husband's name
- Mother's name
- CNIC number
- Date of birth
- Gender
- Marital status
- Phone numbers (mobile and WhatsApp)
- Addresses (present and permanent)
- Other personal details

#### Step 2: Employment Details

- **Department:** Select from dropdown menu
- **Designation:** Select from dropdown menu
- **Scale Grade:** Select appropriate pay scale
- **Role Tag:** Categorize by role type
- **Employment Type:** Regular, Contract, etc.
- **Effective Dates:** Start and end dates
- **Office Location:** Select location
- **Reporting Officer:** Enter officer's ID

#### Step 3: Education

Click "Add Education" to add qualifications:

- Select education level
- Enter institution name
- Enter year of completion
- Enter marks or GPA (optional)

#### Step 4: Experience (Optional)

Click "Add Experience" to add past work:

- Company name
- Position title
- Start and end dates
- Description (optional)

#### Step 5: Documents

Upload supporting documents:

- CNIC copies
- Educational certificates
- Photos
- Other relevant documents

1. Click "Upload Document"
2. Select file from your computer
3. Enter document name and type
4. Click "Upload"

#### Step 6: Save the Employee

1. Review all information for accuracy
2. Click **"Save"** or **"Create Employee"** button
3. The system will create the employee record
4. You'll be redirected to the employee profile page

## Editing Employee Information

### Accessing the Edit Form

1. Open an employee's profile
2. Click the **"Edit"** button at the top
3. The system will open the edit form

### Making Changes

1. Modify any fields that need updating
2. Upload new documents if needed
3. Update employment details as necessary
4. Click **"Save Changes"** when done

### Updating Employment Status

To change employment status:

1. Go to employee profile
2. Navigate to employment section
3. Click "Update Employment"
4. Change status (Active/Inactive/Terminated)
5. Save changes

---

## Managing Employment Records

### Adding New Employment Record

Employees may have multiple employment records if they change positions:

1. Open employee profile
2. Go to "Employment History" section
3. Click "Add New Employment"
4. Fill in the new employment details:
   - Department
   - Designation
   - Effective dates
   - Type of employment
   - Office location
5. Click "Save"

### Updating Employment Details

1. Find the employment record to update
2. Click "Edit"
3. Modify the necessary fields
4. Save changes

### Employment History

The system maintains a complete history of employment changes:

- Transfers
- Promotions
- Designation changes
- Department changes
- Salary adjustments
- Status changes

All history is recorded with:

- Date of change
- User who made the change
- Old value and new value
- Reason for change (if provided)

## Managing Documents

### Uploading Employee Documents

1. Open employee profile
2. Go to "Documents" section
3. Click "Upload Document"
4. Select document category:
   - Personal (CNIC, photos)
   - Educational certificates
   - Employment documents
   - Contracts
   - Other
5. Choose file from computer
6. Enter description (optional)
7. Click "Upload"

### Viewing Documents

1. Scroll to "Documents" section
2. Documents are listed by category
3. Click document name to view
4. Click download icon to save file

### Document Types

Common document categories:

- **CNIC:** Copy of National ID card
- **Educational Certificates:** Degrees and diplomas
- **Contracts:** Employment agreements
- **Photos:** Profile pictures
- **Other:** Miscellaneous documents

## Understanding Employee Status

- Currently employed

- Included in attendance tracking

### Inactive Employees

- Suspended
- Temporarily assigned elsewhere

### Deleted/Removed Employees

- Terminated or resigned
- Soft-deleted from system

## Troubleshooting

### Cannot Find Employee

- Try searching by partial name
- Check if using correct search term
- Verify employee is not deleted
- Contact system administrator

### Cannot Edit Employee

- Check your permissions
- Ensure you have "Update Employee" access
- Contact administrator if needed

### Document Upload Fails

- Check file size (max size may be limited)
- Verify file format is supported
- Ensure stable internet connection
- Try again with smaller file
